# 行业参考：数码3C / 运动户外 / 母婴

---

## 数码3C / 电子产品

### 风格
**代表感觉**：小米有品、苹果风白牌、数码周边品牌

**背景策略**：
- 产品展示区：纯黑 `#000000` 或纯白 `#FFFFFF`
- 功能介绍区：深灰 `#1A1A1A` 到浅灰 `#F5F5F7` 交替
- 绝对不用暖色，用冷灰或蓝灰色调

**视觉串联**：
- 产品渲染图（科技感光线打底）
- 规格参数表格化展示
- 功能点用图标+短语呈现（类似苹果官网横向滚动）

**模块结构**：
```
① 产品主图（深色背景，高光渲染）
② 核心卖点（3–5点，图标+数字+短语）
③ 外观/配色展示（颜色变体横排）
④ 功能详解（一功能一大图）
⑤ 规格参数（表格）
⑥ 包装清单
```

**排版特点**：
- 字体：无衬线为主（PingFang SC / Noto Sans SC）
- 数字要大：「30W快充」「120Hz」放到 60–80px
- 纯英文产品名用 SF Pro / Inter 感的字体

**CSS参考**：
```css
/* 数码品牌典型配色 */
--bg-dark: #000000;
--bg-light: #F5F5F7;
--text-primary: #1D1D1F;
--text-secondary: #6E6E73;
--accent: #0071E3; /* 苹果蓝 */
/* 或 */
--accent: #FF6900; /* 小米橙 */
```

## 运动 / 户外 / 瑜伽

### A. 专业运动品牌（跑步/健身）
**背景策略**：
- 主背景：纯黑 `#050505` 或纯白
- 强调色：品牌色（荧光绿/电光橙/钴蓝）
- 运动场景图：高饱和，对比度拉高

**视觉串联**：
- 运动轨迹线（SVG路径，品牌色）
- 运动员动态大图（带模糊/速度感）
- 配速/距离/心率等数字大字

**模块结构**：
```
① 运动大图（全出血，动感构图）
② 技术卖点（透气/支撑/回弹等）
③ 应用场景（跑步/健身/球场）
④ 产品系列
⑤ 数据背书（减震测试/耐磨数据）
```

---

### B. 户外/露营
**背景策略**：
- 主背景：`#F0EBE0`（沙漠米）或 `#E8EDE8`（原野绿白）
- 场景图：山地/森林/草原，自然光
- 深色区：`#1A2318`（深森林绿）

**视觉串联**：
- 地形轮廓线（山峰剪影 SVG）
- 露营装备平铺俯拍（knolling 排列）
- 海拔/温度等户外数据

---

### C. 瑜伽/普拉提
**背景策略**：
- 主背景：`#F8F4EF`（皮肤米）
- 辅色：`#D4C5B5`（裸粉）或 `#B5C4B5`（尘绿）
- 模特图：浅色/无背景 studio 风格

**视觉串联**：
- 瑜伽体式线条图（金色或品牌色线条）
- 织物纹理细节（面料质感特写）

## 母婴 / 儿童

### 风格
**代表感觉**：babycare、贝亲、国内小众母婴品牌

**背景策略**：
- 主背景：`#FFFEF9`（米白）或 `#F5F9FF`（浅蓝白）
- 辅色：`#FFE8D6`（奶橙）/ `#D6EBFF`（天蓝）/ `#D6F5D6`（嫩绿）
- 绝对避免深色，保持明亮轻松

**视觉串联**：
- 圆角色块（母婴行业圆角比其他行业大 2–3倍）
- 婴儿/儿童的笑脸照片
- 可爱插画元素（动物/星星/云朵）

**模块结构**：
```
① 亲子场景大图（温馨氛围）
② 安全/认证信息（很重要，消费者决策关键）
③ 核心卖点（安全材质/人体工学/易清洗）
④ 适龄说明（0–3月/3–6月/6–12月）
⑤ 产品系列
⑥ 妈妈好评展示
```

**排版特点**：
- 圆润字体（思源圆体或类似风格）
- 字号稍大，行高宽松（line-height 2+）
- 安全认证图标要显眼
- 避免密集文字，以图为主

**CSS参考**：
```css
--bg: #FFFEF9;
--accent-warm: #FFB085; /* 暖橙 */
--accent-cool: #85C1E9; /* 天蓝 */
--text: #2C2C2C;
--text-soft: #888888;
border-radius: 16px; /* 所有卡片圆角更大 */
```

## 关键差异矩阵

| 行业 | 背景色调 | 核心卖点 | 数字大字 | 圆角 | 人物图比例 |
|------|---------|---------|---------|------|---------|
| 数码3C | 纯黑/纯白 | 参数/性能 | 必须有 | 0 | 低 |
| 专业运动 | 深黑/品牌色 | 技术/性能 | 有 | 小 | 高（动态）|
| 户外露营 | 自然色 | 功能/耐用 | 无 | 小 | 中（风景）|
| 瑜伽 | 裸色/尘色 | 质感/舒适 | 无 | 中 | 高（体式）|
| 母婴 | 明亮暖白 | 安全/认证 | 无 | 大 | 极高（亲子）|
