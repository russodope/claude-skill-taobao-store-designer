# 行业参考：美妆 / 护肤 / 香氛

## 风格分类

### A. 高端护肤/精华类
**代表感觉**：PMPM、溪木源、HBN

**背景策略**：
- 主背景：`#FDFBF8`（象牙白）或 `#F8F4EE`（奶油白）
- 产品区：纯白底，无任何纹理
- 成分/功效区：`#EFF5EE`（薄荷浅绿）或 `#F0EBF5`（薰衣草浅紫）

**视觉串联**：
- 产品瓶身的曲线提取为 SVG 装饰线
- 成分植物插画（简笔，单色）贯穿各区块

**排版特点**：
- 强调功效数字：「28天」「3倍」「99%」用超大字号（60–80px）展示
- 中英文并行，英文是成分/品牌名
- 科学感：数据表格、成分表、效果对比

**模块结构**：
```
① 主打产品大图（产品+场景/模特）
② 核心功效 3 点（图标+大数字+短语）
③ 成分故事（插画+段落文字）
④ 系列商品（2列，白底）
⑤ 用户晒单/评价展示
⑥ 品牌背书（实验室/检测报告风格）
```

---

### B. 彩妆/口红/眼影
**代表感觉**：花西子、橘朵、酵色

**背景策略**：
- 花西子风：`#F2E8D8`（金粉米）+ `#2D1A0E`（深棕）
- 橘朵/年轻款：`#FFF0F3`（蜜桃粉）或品牌色渐变
- 大胆用渐变：`linear-gradient(135deg, #FFB3C6, #FF6B9D)`

**视觉串联**：
- 产品的色号展示条（从左到右横向渐变色块）
- 口红/眼影的质地特写图过渡
- 色系名（大地色/玫瑰色/珊瑚色）作为区块标题

**排版特点**：
- 色卡横排展示色号（非常重要，买家决策关键）
- 试色图（上妆效果）放大展示
- 节日/季节氛围感强（圣诞/春日/限定款）

---

### C. 香氛/蜡烛/香薰
**代表感觉**：观夏、MAISON MARGIELA风小众香

**背景策略**：
- 主背景：`#F0EBE3`（沙色）或 `#1C1A17`（暗夜）
- 产品多用暗色摄影（烛光/低调光源）
- 绝对避免鲜艳纯色背景

**视觉串联**：
- 蜡烛火焰、烟雾、光晕的视觉意象
- 香调结构（前调/中调/尾调）用类似乐谱的排版呈现
- 场景图：书桌/浴室/卧室，统一温暖低饱和度色调

**排版特点**：
- 香调名用诗意化命名（比如「寒露」「雨后青苔」）
- 大量留白，产品图居中漂浮
- 英文介绍与中文并排，Cormorant Garamond italic

## 关键差异点 vs 珠宝行业

| 维度 | 珠宝/首饰 | 美妆/护肤 |
|------|---------|---------|
| 背景 | 暖亚麻 | 奶油白/品牌色渐变 |
| 核心卖点展示 | 工艺/材质/故事 | 功效数字/成分/色号 |
| 图片主角 | 产品+模特局部 | 产品特写+上妆效果 |
| 文字风格 | 诗意/东方意境 | 科学感/功效化/情绪化 |
| 必备模块 | 系列展示 | 功效区块、色号展示、评价 |
| 颜色鲜艳度 | 低，素雅 | 可以很高（彩妆）|
